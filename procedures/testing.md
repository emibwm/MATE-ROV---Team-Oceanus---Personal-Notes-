# Testing Procedures
This document outlines the testing protocols for the Team Oceanus ROV systems.

## Pre-Dive Testing
- [ ] Visual inspection of all seals.
- [ ] Battery voltage check.
- [ ] Tether integrity check.

## System Checks
- [ ] Motor functionality.
- [ ] Sensor calibration and data stream.
- [ ] Control surface/thruster responsiveness.

## Post-Dive Analysis
- [ ] Data log review.
- [ ] Component wear inspection.
