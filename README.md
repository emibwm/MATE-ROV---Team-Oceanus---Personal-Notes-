# MATE-ROV - Team Oceanus

## Project Overview
Team Oceanus is an aspiring participant of the underwater robotics MATE ROV competition. This repository contains personal notes and documentation to guide the team towards better systems engineering practices and organizational efficiency.

## Repository Structure
- `.github/`: Issue templates and GitHub workflows.
- `docs/`: Primary documentation including safety, faq and getting started guides.
- `procedures/`: Step-by-step assembly and fabrication guides.
- `CONTRIBUTING.md`: Team participation guidelines.
- `README.md`: Project overview (this file).
